\# Java Loan Calculator



A simple command-line loan calculator written in Java.



This project calculates the monthly payment of a loan based on:

\- Loan amount

\- Annual interest rate

\- Loan duration in years



\## How it works



The program uses the standard loan amortization formula to compute the monthly payment and formats the result as currency.



\## Technologies

\- Java (JDK 17+)

\- Command Line (CLI)



\## Project structure

src/

└── com/

└── gabriel/

└── finance/

└── Main.java



\## How to run



Compile: javac src/com/gabriel/finance/Main.java



Run: java -cp src com.gabriel.finance.Main



\## Example



Enter the loan amount: 100000

Enter the annual interest rate (in %): 6

Enter the number of years: 30

The monthly payment is $599.55
